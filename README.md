# Music Toolkit for Producers & Musicians

A powerful web platform built for **music producers and musicians** to simplify their workflow using modern web tech + AI.

From **audio splitting** to **instrument tuning** and **platform-based mastering**, this toolkit is designed to bring studio-grade utilities directly to your browser.

---

## Features

- **Audio Stem Separation**
  - Split songs into vocals, drums, bass, and more using AI (Demucs)

- **Instrument Tuner**
  - Real-time tuning for instruments like guitar, bass, etc.

- **Platform-Based Mastering**
  - Optimize your tracks for platforms like YouTube, Instagram

- **Audio Visualization**
  - Waveform-based visual feedback

- **More Features Coming Soon**
